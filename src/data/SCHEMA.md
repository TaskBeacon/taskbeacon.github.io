# Task Index Schema

`src/data/tasks_index.json` is generated by `scripts/build-index.mjs`.

Schema version: `2`

## Top-level

- `schema_version`: number
- `generated_at`: ISO timestamp string
- `org`: string (GitHub org)
- `tasks`: array of task objects

## Task object

Required:

- `repo`: string (repo name)
- `full_name`: string (`Org/Repo`)
- `html_url`: string
- `default_branch`: string
- `short_description`: string
- `tags`: object
  - `paradigm`: string[]
  - `response`: string[]
  - `modality`: string[]
  - `language`: string[]
- `keywords`: string[]
- `last_updated`: ISO timestamp string

Optional:

- `maturity`: string | null (e.g. `smoke_tested`, `piloted`)
- `psyflow_version`: string | null
- `has_voiceover`: boolean | null
- `structure`: object
  - `config`: boolean
  - `assets`: boolean
  - `src`: boolean
- `readme_run_anchor`: string (e.g. `#run`)

Notes:

- `short_description` is taken from `task.yaml.short_description` when present; otherwise it falls back to the README `Short Description` table row, then the README first paragraph.
- `maturity` is taken from `task.yaml.maturity` when present; otherwise it is parsed from README badges like `![Maturity: smoke_tested]`.
