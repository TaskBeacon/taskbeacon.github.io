# Task Index Schema

`src/data/tasks_index.json` is generated by `scripts/build-index.mjs`.

Schema version: `1`

## Top-level

- `schema_version`: number
- `generated_at`: ISO timestamp string
- `org`: string (GitHub org)
- `tasks`: array of task objects

## Task object

Required:

- `repo`: string (repo name)
- `full_name`: string (`Org/Repo`)
- `html_url`: string
- `default_branch`: string
- `short_description`: string
- `tags`: object
  - `paradigm`: string[]
  - `response`: string[]
  - `modality`: string[]
  - `language`: string[]
- `keywords`: string[]
- `last_updated`: ISO timestamp string

Optional:

- `psyflow_version`: string | null
- `has_voiceover`: boolean | null
- `structure`: object
  - `config`: boolean
  - `assets`: boolean
  - `src`: boolean
- `readme_run_anchor`: string (e.g. `#run`)
